# TypeScript Hello World

## Project Overview

A simple command-line program written in TypeScript that outputs a greeting to the console to verify TypeScript environment setup and execution.

## User Stories

### US-001: Output Greeting

**Priority**: 1
**Status**: pending

**User Story**: As a user, I want the program to print 'Hello, World!' to the console, so that I know the TypeScript compilation and execution environment are working.

#### Acceptance Criteria

- [ ] 1. WHEN the program is executed, THE system SHALL output the exact text 'Hello, World!' to standard output.
- [ ] 2. IF the execution is successful, THEN THE system SHALL exit with status code 0.
